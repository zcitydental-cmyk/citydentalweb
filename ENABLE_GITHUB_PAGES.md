# How to Enable GitHub Pages for City Dental Website

## Why You're Seeing 404 Error

Your website files are on GitHub, but **GitHub Pages is not turned on**. GitHub Pages is a free hosting service that needs to be manually enabled in your repository settings.

## Step-by-Step Instructions

### Step 1: Sign In to GitHub
1. Go to https://github.com
2. Click "Sign in" (top right)
3. Enter your credentials for account: **zcitydental-cmyk**

### Step 2: Go to Your Repository
1. After signing in, go to: https://github.com/zcitydental-cmyk/citydentalweb
2. You should see your 3 files: index.html, styles.css, logo.jpeg

### Step 3: Open Settings
1. Click the **"Settings"** tab at the top of the page (it's next to "Insights")
2. If you don't see Settings, you may need to scroll the top menu to the right

### Step 4: Find Pages Section
1. In the left sidebar, scroll down
2. Look for **"Pages"** under the "Code and automation" section
3. Click on **"Pages"**

### Step 5: Enable GitHub Pages
1. Under **"Build and deployment"** section
2. Find **"Source"** dropdown
3. Select **"Deploy from a branch"**
4. Under **"Branch"**:
   - First dropdown: Select **"main"**
   - Second dropdown: Select **"/ (root)"**
5. Click the **"Save"** button

### Step 6: Wait for Deployment
1. The page will refresh
2. You'll see a message: "GitHub Pages source saved"
3. Wait 1-3 minutes for the site to build
4. Refresh the Settings > Pages page
5. You should see a green box that says: **"Your site is live at https://zcitydental-cmyk.github.io/citydentalweb/"**

### Step 7: Visit Your Live Website
Open this URL in your browser:
```
https://zcitydental-cmyk.github.io/citydentalweb/
```

## Troubleshooting

### If you still see 404 after enabling:
- Wait 5 minutes and try again (deployment takes time)
- Clear your browser cache (Ctrl+Shift+Delete)
- Try opening in incognito/private mode

### If you can't find Settings tab:
- Make sure you're signed in to the correct account
- You must be the owner of the repository to see Settings

### If Pages option is not in Settings:
- Make sure the repository is public (not private)
- Check that you have admin access to the repository

## What Happens After Enabling

Once enabled, your website will be:
- ✅ Live at: https://zcitydental-cmyk.github.io/citydentalweb/
- ✅ Accessible to anyone with the link
- ✅ Automatically updated when you push changes to GitHub
- ✅ Free hosting with HTTPS
